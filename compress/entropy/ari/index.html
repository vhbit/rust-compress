ari

Module compress::entropy::ari
[-]  [+] [src]

Arithmetic encoder/decoder using the Range encoder underneath. Can be used in a general case of entropy coding stage. Supposed to be fast.

Links

http://en.wikipedia.org/wiki/Arithmetic_coding http://en.wikipedia.org/wiki/Range_encoding

Example

use std::io::{MemWriter, MemReader};
use compress::entropy::ari;

// Encode some text
let text = "some text";
let mut e = ari::ByteEncoder::new(MemWriter::new());
e.write_str(text);
let (encoded, _) = e.finish();

// Decode the encoded text
let mut d = ari::ByteDecoder::new(MemReader::new(encoded.unwrap()));
let decoded = d.read_to_end().unwrap();
